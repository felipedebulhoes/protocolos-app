# Scripts de uso único (histórico)

Estes scripts Python foram usados pontualmente para popular/migrar conteúdo
(protocolos, imagens, certificados, vídeos) durante o desenvolvimento inicial
do app. Não fazem parte do build nem rodam em produção — ficam aqui apenas
como referência/histórico. Antes de rodar qualquer um deles de novo, releia
o conteúdo: vários assumem caminhos locais específicos da máquina onde foram
escritos originalmente.
